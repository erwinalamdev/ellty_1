# Page Selector Component

A modern, responsive page selector UI component built with React + TypeScript + Vite.

**Author:** Erwin Alam Syah Putra

## Features

- Select/deselect individual pages
- "All pages" checkbox to select/deselect all at once
- Hidden scrollbar with scroll functionality
- Smooth hover and click animations
- Clean, modern UI design

## Tech Stack

- **React** - UI Library
- **TypeScript** - Type safety
- **Vite** - Build tool & dev server
- **Tailwind CSS** - Styling

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
